Bulaga

About

Bulaga is an open-source javascript library using jQuery that shows a content with effect when the page is scrolled. Bulaga is the Filipino version of Peek-a-boo!

Prerequisite

  • jQuery

How to use

Using a normal DOM selector in jQuery, call the bulaga() function.

$(selector).bulaga(options);

You may also add some options depending on your preferences. _options_ is in JSON format. If you want to set the duration of the animation, you can do...

$(selector).bulaga({"duration": 5000});

Options

  • animation

    Sets the type of animation you want for the element

    type String
    values "FADE_IN", "SLIDE_LEFT", "SLIDE_RIGHT", "SLIDE_UP", "SLIDE_DOWN"
    default "FADE_IN"
  • duration

    The duration of animation in milliseconds you want for the element.

    type Integer
    default 500
  • position

    The percentage of your screen you want your element to show relatively to the original position of your element. Basically, the higher the number, the sooner the element will show.

    type Integer
    values 0-1
    default .25
  • bounce

    Option for a bounce effect.

    type Boolean
    default false
  • distance

    The starting distance from the original position.

    type Integer
    default 100
  • base

    The css attribute to be used for animation. This can only be used if the element's position is absolute or fixed. If false, the css attribute to be used depends on the direction of your slide animation.

    type String
    values "top", "bottom", "left", "right"
    default false
  • repeat

    The animation will repeat if scrolled again.

    type Boolean
    default false
  • callback

    Things you want to do after the animation has ended.

    type Function
    default false
